# Монорепозиторий

Живой пример: https://gkit.pages.dev/

## Разработка

в app/package.json поменять на

`"@itgenio/gkit": "file:../gkit",`

и

`npm i`

## Релиз пакета

`npm version patch && npm publish`

## Описание

`app` - демонстрационное приложение.

`gkit` - библиотека компонентов.

`icons` - библиотека иконок.
